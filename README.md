# NotesApp - Android SQLite/Room Project

Това е мобилно приложение за Android, създадено за целта на курсова работа. Приложението демонстрира работа с локална база данни SQLite чрез библиотеката **Room**.

### Функционалности:
*   Въвеждане на текстови бележки.
*   Запис на данните в локална база данни.
*   Визуализация на записите в списък (ListView).
*   Автоматично обновяване на интерфейса при нов запис.

### Технологии:
*   **Language:** Kotlin
*   **Database:** Room (SQLite)
*   **Processing:** KSP (Kotlin Symbol Processing)
